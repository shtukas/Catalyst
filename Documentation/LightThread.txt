LightThread { 
    :uuid        : String
    :unixtime    : Integer
    :description : String
    :priorityXp  : LightThreadPriorityXP
    :status      : LightThreadStatus
    :done        : Array[LightThreadTimeRecordItem]
    :streamuuid  : String UUID
}

LightThreadPriorityXP
    ["interruption-now"]
    ["must-be-all-done-today"]
    ["stream-important", <commitment in hours>]
    ["stream-luxury", <commitment in hours>]

LightThreadStatus
	["paused"]
	["running-since", Unixtime]

LightThreadTimeRecordItem: {
	"uuid"     => not particularly relevant, random
	"unixtime" => Unixtime
	"timespan" => Integer, in seconds
}

LightThreadTimeRecordItem(s) are stored in a Iphetra set of uuid "<LightThreadUUID>:DFB99806"
