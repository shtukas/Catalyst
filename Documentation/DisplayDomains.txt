
DisplayDomainClaim (
    claimtime  : Integer # unixtime
    objectuuid : String
    domainname : String
)

DisplayDomainData: (
	"claims" :        Map[String # objectuuid, DisplayDomainClaim ] 
	"active-domain" : String
)

The mapping has the consequence that an object can only be associated to one particular claim, this prevents an object to be in more than one domain.
