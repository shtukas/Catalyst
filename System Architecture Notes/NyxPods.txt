NyxPods are named folders that are reference points in the nyx graph. They are data point types (replacing the old unique names and the biggest aion points). The notion of aion point if not going to disappear as it will still be useful for transcient file tree

A NyxPod is any folder that has a name which contains the a string of the form NyxPods-<UUID>, for instance: NyxPods-fcb5541a-6e2e-48dd-9981-2f137fe8490c
