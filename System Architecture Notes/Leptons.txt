Lepton database

The filename is a Sqlite3 file.

table: lepton
    _key_      text
    _value_    data

create table lepton (_key_ text, _value_ data);

-------------------------------------------------------------------------

Lepton common

    ("18da4008-6cb2-4df0-b9d5-bb9e3b4f949a", <type>: String)

    ("9fb612ab-698c-4f6a-ab99-5aadb3f727d0", <description>: String) 
        The description is optional. If present it is returned when a lepton description is asked for
        otherwise the type / contents are introspected.
