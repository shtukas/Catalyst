
Nyx

For history and description, see NyxCore.

The two required keys that a NyxObject must have are:
    uuid             : String
    nyxType          : String
    creationUnixtime : Float # unixtime with decimals

------------------------------------------------------

    "uuid"             : String

    "passenger"        : TimePodPassenger

            "type"        : "description"

Wave
{
    "uuid"             : String
    "nyxType"          : "wave-12ed27da-b5e4-4e6e-940f-2c84071cca58"
    "creationUnixtime" : Float # Unixtime with decimals
    "description"      : String
    "schedule"         : Schedule
}
    Schedule
    {
        "uuid"         => SecureRandom.hex,
        "@"            => "sticky"
        "from-hour"    => fromHour
    }
    {
        "uuid"         => SecureRandom.hex,
        "@"            => type,
        "repeat-value" => value
    }
