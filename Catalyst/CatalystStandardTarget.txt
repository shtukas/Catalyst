
A CatalystStandardTarget is an object of the form

```
{
    "type"       : "line"
    "line"       : String
}
{
    "type"       : "url"
    "url"        : String
}
{
    "type"       : "file"
    "filename"   : String (CoreData managed)
}
{
    "type"       : "folder"
    "foldername" : String (CoreData managed)
}
{
    "type"       : "unique-name"
    "uniquename" : String # Use Atlas to find them
}
{
    "type"          : "directory-mark"
    "directorymark" : String # Use Atlas to find them.
                             # Atlas also supports findind directory marks.
                             # `Nyx-Directory-Mark.txt` files with a unique name inside.
}
```
