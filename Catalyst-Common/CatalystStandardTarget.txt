
A CatalystStandardTarget is an object of the form

```
{
    "type"       : "line"
    "line"       : String
}
{
    "type"       : "file"
    "filename"   : String
}
{
    "type"       : "url"
    "url"        : String
}
{
    "type"       : "folder"
    "foldername" : String
}
```

The file and the folder are managed by CoreData.
